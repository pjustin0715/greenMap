# greenMap

A smart campus navigation app

## Features

- Gives you the shortest route between buildings/landmarks
- Shows total distance from the destination
- Trip calculation
- Tells you buildings/landmarks along the way so you dont get lost

## Tech Stack

- Python
- Streamlit
- NetworkX

## install
- streamlit
- networkx
- pandas
- plotly
- Pillow

```bash
streamlit run app.js
```
